# 문서 스타일
- 규칙은 "한다/않는다" 서술형으로 통일한다.
- 불렛은 `*` 대신 `-`로 통일한다.
- 불렛이 없는 줄(ex. 독립 텍스트)에 `-`를 추가하되, 기존 숫자 불렛(`1.`, `2.` 등)은 그대로 유지한다.
- 섹션 헤더(`##`, `###`, `####`) 다음에 빈 줄 없이 바로 내용을 작성한다.
- 줄바꿈은 빈 줄 대신 trailing space(뒤 띄어쓰기)를 사용한다.
- 섹션 구분은 `---` 대신 빈 줄 + `<br>` + 빈 줄로 처리한다.
- 파일 마지막 줄에 빈 줄을 두지 않는다.
- 구조 트리의 주석은 핵심만 간결하게 작성한다. 부가 설명(예: API 1회, ID + 기본 정보 등)은 붙이지 않는다.
- 설정 안내는 설명 텍스트 한 줄 + 코드블록으로 작성한다. 코드블록 내 주석은 최소화한다.
