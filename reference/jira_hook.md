# Jira Hook 규칙
Jira 연동 프로젝트에서 적용한다. commit-msg hook이 있으면 자동 변환된다.

## Hook 동작
- 브랜치 이름에서 티켓 번호를 추출한다 (예: `feat/PROJ-7` → `PROJ-7`).
- prefix를 `[prefix]`으로 자동 변환한다.
- 티켓 번호를 커밋 메시지에 자동 삽입한다.

## 작성 규칙
- hook이 있으면 자동 변환되므로 prefix만 소문자로 작성한다. 없으면 `[prefix]` 형식으로 직접 작성한다.
- 지라 번호는 반드시 커밋 본문에 넣는다.
- 커밋 메시지 형식:
```
[prefix] 내용
- 상세 내용
- 상세 내용
- 지라 티켓 번호
```
