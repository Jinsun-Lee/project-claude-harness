# 레포 명명 규칙
- 구조: `{카테고리}-{주제}[-{구분자}]`
- `kebab-case`로 작성한다.
- 카테고리 접두어:
  - `study`: 계속 관리하는 것, 계속 참고하는 템플릿같은 것
  - `lecture`: 수강하는 과목, 기간이 정해져 있음
  - `project`: 단기간으로 진행한 프로젝트, 일부 계속 관리하는 것이나 템플릿이 있을 수 있음
  - `misc`: 그해에 작업한 이것저것, 뒤에 년도만 붙임 (`misc-23`, `misc-24`, `misc-25`, `misc-26`)
  - `ros2_`: ROS2 패키지, `snake_case`로 작성
- 주제: 레포가 무엇에 대한 것인지 나타내는 핵심 도메인.
- 구분자: 같은 주제를 분리할 때만 사용 (`-private`, `-template`).
